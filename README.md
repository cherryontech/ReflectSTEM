## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/cherryontech/unmatched.git
2. Navigate to the project directory:
   ```bash
   cd unmatched
3. Install the dependencies:
   ``` bash
   npm install
4. Start the development server:
   ``` bash
   npm run dev
